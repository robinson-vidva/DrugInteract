Check drug interactions

Enter two or more drugs. Each name is normalized with RxNorm, then this tool reads the live U.S. FDA drug labels (openFDA) and tells you whether one drug's label text mentions the other. It does not grade severity.

Suggestions from RxNorm appear as you type; you can also type any name freely. Brand names work too (e.g. Coumadin).
